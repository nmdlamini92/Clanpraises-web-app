# ClanPraises web-app

An interactive library/database of clanpraises

## Get started

1. Go inside server folder 

  cd server

2. Install dependencies 

   ```bash
   npm install
   ```

3. Start the server

   ```bash
   npm start
   ```
4. Return to root folder

   cd ..

5. Install dependencies 

   ```bash
   npm install
   ```

3. Start next app

   ```bash
   npm run dev
   ```
