# ClanPraises web-app

An interactive library/database of clanpraises

## Get started

Enter server folder then;

1. Install dependencies 

   ```bash
   npm install
   ```

2. Start the server

   ```bash
   npm start
   ```
Return to root folder then;

3. Install dependencies 

   ```bash
   npm install
   ```

3. Start next app

   ```bash
   npm run dev
   ```
